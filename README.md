# LiqoApp

Mobile app for liqo-js.

## Installation

To install all dependencies,

```sh
$ git clone https://github.com/berkedel/liqoapp.git
$ cd liqoapp
$ npm install
$ react-native link
```

## Run the app

For Android user,

```sh
$ react-native run-android
```

For iOS user,

```sh
$ react-native run-ios
```
